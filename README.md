# Bangla Web Fonts CDN

A beautiful, modern CDN for Bangla web fonts. Instantly import, embed, and preview Bangla fonts for your web projects.

## Fonts Available
- **BCC Purno**
- **Hadi**
- **July**

---

## Usage

### 1. Import via CSS
```css
@import url('https://fonts.sadi.com.bd/fonts/bcc-purno/font.css');
@import url('https://fonts.sadi.com.bd/fonts/hadi/font.css');
@import url('https://fonts.sadi.com.bd/fonts/july/font.css');
```

### 2. Embed via HTML
```html
<link rel="stylesheet" href="https://fonts.sadi.com.bd/fonts/bcc-purno/font.css">
<link rel="stylesheet" href="https://fonts.sadi.com.bd/fonts/hadi/font.css">
<link rel="stylesheet" href="https://fonts.sadi.com.bd/fonts/july/font.css">
```

### 3. CSS Font Family
```css
font-family: 'BCC Purno', sans-serif;
font-family: 'Hadi', sans-serif;
font-family: 'July', sans-serif;
```
