Quick Start

Embed Font

To use a font on your website, embed this code in the <head> of your HTML document.

STANDARD

<link href="https://fonts.sadi.com.bd/fonts/bcc-purno/font.css" rel="stylesheet">

IMPORT in CSS

@import url('https://fonts.sadi.com.bd/fonts/bcc-purno/font.css');

Specify in CSS

Use the following CSS rule to specify the font-family:

font-family: 'BCC Purno', sans-serif;

Example

body {
  font-family: 'BCC Purno', sans-serif;
}
