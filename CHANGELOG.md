# Changelog

All notable changes to Bangla Web Fonts CDN will be documented in this file.

## [1.0.0] - 2026-01-01

### Added
- Initial release of Bangla Web Fonts CDN
- Three Bengali fonts:
  - BCC Purno (WOFF2, WOFF formats)
  - Hadi (TTF format, includes Rounded variant)
  - July (TTF format with Regular, Italic, Bold, Bold Italic)
- Professional website with Tailwind CSS
- Individual font pages with previews and usage instructions
- Interactive JavaScript features:
  - Copy-to-clipboard functionality
  - Smooth scrolling
  - Scroll-to-top button
  - Animated feature cards
- Complete documentation:
  - README.md with quick start guide
  - DEPLOYMENT.md with GitHub Pages setup
  - CONTRIBUTING.md for contributors
  - SECURITY.md for security policies
- SEO optimization:
  - sitemap.xml
  - robots.txt
  - Meta tags and descriptions
- GitHub Actions workflow for automatic deployment
- Custom domain support (CNAME file)
- 404 error page
- License files
