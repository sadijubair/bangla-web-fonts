# Security Policy

## Reporting Security Issues

If you discover a security vulnerability in Bangla Web Fonts CDN, please report it by:

1. **DO NOT** open a public GitHub issue
2. Email the maintainer directly (if contact provided)
3. Or use GitHub's private security advisory feature

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| Latest  | :white_check_mark: |

## Security Best Practices

When using this CDN:

1. Always use HTTPS links to load fonts
2. Verify the integrity of font files
3. Keep your dependencies up to date
4. Report any suspicious activity

## Updates

Security updates will be released as soon as possible after a vulnerability is confirmed.

Thank you for helping keep Bangla Web Fonts CDN secure!
