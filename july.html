Embed Font

To use this font on a website, embed this code in the <head> of your HTML document.

STANDARD

<link href="https://fonts.sadi.com.bd/fonts/july/font.css" rel="stylesheet">

IMPORT in CSS

@import url('https://fonts.sadi.com.bd/fonts/july/font.css');

Specify in CSS

Use the following CSS rule to specify this font-family:

font-family: 'July', sans-serif;

Example

body {
  font-family: 'July', sans-serif;
}

.regular {
  font-weight: 400;
}

.bold {
  font-weight: 700;
}

.italic {
  font-style: italic;
}
